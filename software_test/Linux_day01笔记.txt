1.操作系统概念
  操作系统是一种特殊的软件，安装在电脑电脑上的第一层软件


2.作用
操作系统作用：
相当于电脑的大管家
管理计算机硬件和软件资源
向下控制计算机的硬件
向上管理计算机的软件资源


3.应用领域
桌面应用领域
服务器应用领域
嵌入式应用领域
移动设备应用领域

Linux主要用于 服务器  嵌入式 移动设备
windows 桌面应用领域


虚拟机：
通过软件模拟的具有完整硬件环境运行的平台
作用： 可以在虚拟机中运行操作系统（操作系统镜像）
特点： 运行在虚拟机中系统和当期电脑上的操作系统不会相互影响

linux内核版
内核版是操作系统最核心的部分， 管理计算机软件和硬件程序最核心的部分
（比如 汽车的发动机和车轮以及 底盘）

linux发行版
发行版就是在内核版的基础之上添加了很多的配套的应用程序
桌面应用，字体库， 数据库， 办公套件 
linux发行版 = linux内核版 + 桌面程序 + 字体库等配套的程序

Linux系统下特殊目录:
1./ 根目录, 根目录下挂载很多的目录
2./bin 可执行二进制文件的目录 （ls cd )
3./etc 系统配置文件的目录
4.家目录
  /home/admin  普通用户的家目录(放在/home/目录下)  (Linux操作系统是多任务多用户的操作系统)
  /root  系统管理员的家目录(直接放在 根目录下的 )


学习Linux命令的原因：
1.linux操作系统诞生时就没有图形界面，都是通过命令操作的
2.远程链接服务器也是通过Linux命令
3.图形界面操作系统效率没有命令效率高 
4.工作中需要使用到命令（必须会）


Linux常用的命令1：
    我是谁？   whoami  作用：查看当前用户
    我在哪？   pwd     作用：查看当前所在的目录
    我周围有什么？  ls  作用：查看当前目录下的内容
    我要去哪？  cd     作用： 切换工作路径
Linux常用的命令2：
    创建文件   touch 文件名   （创建一个文件）
              touch 文件名1 文件名2 文件名3 （创建多个文件）
    创建文件夹  mkdir 文件夹名
    删除文件   rm 文件名
              rm 文件名1 文件名2 文件名3
    删除文件夹  rm  -r 文件夹名
    清空屏幕    clear

终端命令的格式：
1.命令（必须有）
2.选项（可以没有）
3.参数（可以没有）
4.空格（必须有）

基本格式：
命令 [选项] [参数]

查看命令帮助信息的方式：
1.命令  --help
2.man  命令

  操作方法：
  enter键   一行一行翻阅
  空格      一屏一屏翻阅
  f键       一屏一屏翻阅
  b键       一屏一屏翻阅之前
  q键       退出

1.查看当前目录下的内容：
  ls [-alh] [文件/目录]
  -a 显示指定目录下所有的文件和文件夹（包括隐藏文件，以点开头的文件）
  -l 以列表的方式显示数据
  -h 配合-l以人性化的方式显示数据（显示文件大小）

  . 当前目录
  .. 上一级目录

 ls查看当前目录下的内通配符
 ls *.txt        * 匹配任意多个字符
 ls ?.txt        ？ 匹配任意一个字符
 ls [12345].txt  匹配方括号中出现的任意一个字符
 ls [1-5].txt    匹配方括号中出现的任意一个字符，表示范围



cd命令说明 -- 切换目录
cd   切换到当前用户的主目录(/home/用户目录/)
cd ~ 切换到当前用户的主目录(/home/用户目录/)
cd . 切换到当前目录
cd ..  切换到上一层目录
cd -  切换到上一次目录(回溯)


相对路径和绝对路径:
1.相对路径,路径前面不是以/开始的, 相对于当前的路径
2.绝对路径,路径前面是以 /开始的, 从根目录开始一层一层向下


创建文件:
touch 文件名
1.文件不存在,新建
2.文件存在,更新文件的时间
3.创建多个文件 
  touch 文件1 文件2 文件3 
  touch {a..e}.txt 创建 a.txt b.txt c.txt d.txt e.txt


创建目录:
mkdir 目录名
1.目录名不能与当前路径下目录或文件重名
2.创建多层目录,一步到位  mkdir -p aa/bb/cc

删除命令:
rm [-rf] 文件或目录
1.-f 文件或目录不存在时,忽略提示信息
2.-r 删除目录时,必须要的选项

拷贝操作
1.文件
  cp  源文件  目标文件
  cp  源文件  文件夹/目标文件
2.目录
  cp -r  源目录  目标目录
  cp -r  源目录  文件夹/目标目录
3.-i选项
  不加-i直接覆盖
  加 -i 提示是否覆盖(n不覆盖, y覆盖)
